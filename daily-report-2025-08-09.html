🛒 장바구니 시스템 개발

동적 장바구니 페이지 구현
메인페이지에서 연동되는 장바구니 페이지를 HTML 디자인에 맞춰 동적으로 구현
• /app/shopping-cart/page.tsx 생성
• Zustand 상태관리로 장바구니 데이터 연동
• 브랜드별 상품 그룹핑, 선택/전체선택 기능
• 수량 변경, 삭제 기능 구현
진행 중
장바구니 상태 관리 통합
메인페이지, 상품상세페이지, 장바구니 페이지 간 상태 동기화
• useCartStore Hook으로 전역 상태 관리
• localStorage 연동으로 데이터 영속성 확보
• 실시간 장바구니 개수 업데이트
진행 중

🔧 Next.js 컴포넌트 아키텍처 개선

Server/Client Component 에러 해결
"Event handlers cannot be passed to Client Component props" 오류 수정
• 상품상세 페이지를 Server Component → Client Component 전환
• 인라인 이벤트 핸들러를 React 이벤트 핸들러로 변경
• HTML 문서 구조를 React 컴포넌트 구조로 재작성

⚠️ 발생한 이슈들

• 빌드 시 Server Component 에러 지속 발생
• 프로덕션 환경에서 "문제가 발생했습니다" 에러
• 타입스크립트 변수 선언 순서 문제
• 페이지 라우팅 충돌 문제 (/cart vs /shopping-cart)
진행 중
